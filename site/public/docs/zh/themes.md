# 主题与特效

Mox 内置 **20 套主题**，每套都自带专属的动态背景特效（由 Metal GPU 实时渲染），还有一座会呼吸的「动物岛」活场景。

## 切换主题

- 右上角「主题」打开外观面板（浅色 / 深色分组），点任一主题整窗实时换肤。
- 主题选择会被记住，下次启动沿用。

## 浅色主题（11）

经典、丹枫、紫霭、流光、墨、霁青、日栖、竹，以及三座岛屿：动物岛、樱花岛、雪之岛。

## 深色主题（9）

黛、子夜、德古拉、炉火、霜夜、萤夜、沧海、听雨、绯夜。

## 动态特效

每套主题搭配一种氛围特效：

| 特效 | 代表主题 |
| --- | --- |
| 星砂 / 星空 | 黛、子夜 |
| 极光 | 德古拉、紫霭 |
| 流光 | 流光、绯夜 |
| 浮尘 / 金屑 / 余烬 | 经典、墨、炉火 |
| 樱落 / 枫落 | 日栖、丹枫 |
| 竹风 | 竹 |
| 细雨 / 霜雪 | 听雨、霜夜 |
| 流萤 | 萤夜 |
| 浮泡 | 霁青、沧海 |

## 动物岛活场景

岛屿主题不走粒子特效，而是一整幅 2D 活场景：昼夜循环的天空、流云、飞鸟与繁星；草地上行走互动的小动物（小芽、猫、鸟、兔）；随季节变化的樱花 / 落叶 / 飘雪；还有跟随光标的蝴蝶。

## 强调色与排版

- 「设置 › 外观」可单独挑选**强调色**，作用于链接、按钮与选中态。
- 可调正文字体（衬线 / 无衬线）、字号与行距，标题颜色独立分级。

## 无障碍

- 「设置 › 特效」可关闭动效；或勾选跟随系统「减弱动态效果」，开启后特效与场景将冻结为静帧。
