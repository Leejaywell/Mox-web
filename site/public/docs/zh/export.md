# 导出

把文档导出为多种格式，排版即时可见。

## 内置导出

- **PDF**：可自定义页面大小、页边距、页眉页脚与页码；导出前可在预览中确认排版。
- **HTML**：带样式（内联当前主题）或纯净两种。
- **图片**：把整篇或选区导出为 PNG，适合分享卡片。

## Pandoc 格式

安装 [Pandoc](https://pandoc.org) 后，可导出 Docx、LaTeX、EPUB、reStructuredText 等更多格式。

## 操作

1. 「文件 › 导出」选择目标格式。
2. 在弹出的设置里调整页面 / 样式选项。
3. 选择保存位置，完成。

## 提示

- 导出 PDF 时，代码块、数学公式、图表都会按当前渲染如实保留。
- 图片导出会按屏幕分辨率二倍图输出，更清晰。
