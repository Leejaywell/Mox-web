# 快捷键

## 文件与窗口

| 操作 | 快捷键 |
| --- | --- |
| 新建文档 | `⌘N` |
| 新建标签页 | `⌘T` |
| 新建窗口 | `⌘⇧N` |
| 打开文件夹 | `⌘O` |
| 保存 | `⌘S` |
| 关闭标签页 / 速记 | `⌘W` |
| 切换标签页 | `⌃⇥` |

## 编辑

| 操作 | 快捷键 |
| --- | --- |
| 加粗 | `⌘B` |
| 斜体 | `⌘I` |
| 插入链接 | `⌘K` |
| 撤销 / 重做 | `⌘Z` / `⌘⇧Z` |
| 查找 | `⌘F` |
| 命令面板 | `⌘⇧P` |

## 视图

| 操作 | 快捷键 |
| --- | --- |
| 富文本 / 源码 / 分屏 | 顶部切换 |
| 显示 / 隐藏侧边栏 | `⌘\` |
| 大纲面板 | 编辑区右上角 |

## 速记

| 操作 | 快捷键 |
| --- | --- |
| 新建速记（全局） | `⌥⌘N` |

> 多数快捷键可在「设置 › 快捷键」中自定义。
