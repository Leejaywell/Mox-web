# Shortcuts

## File & windows

| Action | Shortcut |
| --- | --- |
| New document | `⌘N` |
| New tab | `⌘T` |
| New window | `⌘⇧N` |
| Open folder | `⌘O` |
| Save | `⌘S` |
| Close tab / note | `⌘W` |
| Cycle tabs | `⌃⇥` |

## Edit

| Action | Shortcut |
| --- | --- |
| Bold | `⌘B` |
| Italic | `⌘I` |
| Insert link | `⌘K` |
| Undo / Redo | `⌘Z` / `⌘⇧Z` |
| Find | `⌘F` |
| Command palette | `⌘⇧P` |

## View

| Action | Shortcut |
| --- | --- |
| Rich Text / Source / Split | top bar |
| Toggle sidebar | `⌘\` |
| Outline panel | top-right of the editor |

## Quick Notes

| Action | Shortcut |
| --- | --- |
| New quick note (global) | `⌥⌘N` |

> Most shortcuts can be customized in Settings › Shortcuts.
